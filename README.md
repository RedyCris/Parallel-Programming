# 南开大学并行程序设计课程作业

## 项目概述

本仓库为南开大学并行程序设计课程作业的代码与报告集合，主要研究内容为**高斯消去算法与LU分解算法的并行研究**。我们的小组成员通过本项目，深入探讨了如何利用并行计算技术优化经典数值计算算法，以提高计算效率和加速求解速度。

## 研究背景

高斯消去法和LU分解是线性代数中常用的数值计算方法，广泛应用于科学计算、工程分析和数据处理等领域。随着计算需求的增加，传统串行算法在处理大规模数据时的性能瓶颈逐渐显现。因此，研究如何将这些算法并行化，以充分利用多核处理器和分布式计算资源，具有重要的理论和实际意义。

## 仓库内容

本仓库包含以下内容：

- **源代码**: 详细展示了高斯消去算法与LU分解算法的并行实现，涵盖了MPI、OpenMP等并行编程框架的应用。
- **实验报告**: 详细描述了实验设计、实现过程、性能分析和结论。
